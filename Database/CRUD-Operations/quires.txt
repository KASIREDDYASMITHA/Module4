Step 1: Create Table
CREATE TABLE students (
    id SERIAL PRIMARY KEY,
    name TEXT NOT NULL,
    email TEXT UNIQUE NOT NULL,
    age INTEGER,
    course TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
 Output: Table created successfully

Step 4: Update Data
Update the age of one student (example: Rahul Sharma)
UPDATE students
SET age = 23
WHERE name = 'Rahul Sharma';
Output: 1 row updated successfully


Update the course for all students enrolled in one course (example: Web Development → Full Stack Development)
UPDATE students
SET course = 'Full Stack Development'
WHERE course = 'Web Development';
Output: 1 row updated successfully


Step 5: Delete Data
Delete one student using id (example: id = 3)
DELETE FROM students WHERE id = 3;
Output: 1 row deleted successfully


Delete all students below a certain age (example: age < 20)
DELETE FROM students WHERE age < 20;
Output: 1 row deleted successfully (Priya Mehta)
